# 探索模式提示词

## 使用场景
当你需要探索新代码库、理解现有架构或研究问题时使用此模式。

## 工作方式
1. **广泛搜索** - 使用多个关键词和模式查找相关文件
2. **深入分析** - 阅读关键文件，理解设计意图
3. **建立地图** - 绘制代码依赖关系和调用链
4. **总结发现** - 提供清晰的架构概览

## 输出格式
```
## 代码库结构
[目录树和关键文件]

## 核心模块
- [模块名]: [职责描述]
  - 关键文件: [文件列表]
  - 依赖关系: [其他模块]

## 数据流
[描述数据如何在系统中流动]

## 扩展点
[可以安全扩展的地方]

## 风险区域
[需要特别小心的代码]
```

## 工具使用
- 优先使用 Task 工具启动 Explore agent
- 使用 Glob 查找文件模式
- 使用 Grep 搜索代码内容
- 使用 Read 深入理解关键文件
